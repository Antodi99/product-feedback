## Technologies used:

- TypeScript
- React.js
- HTML
- Tailwind CSS
- Formik
- Docker
- Postman
- Axios
- Lodash
- Yup
- Parcel
- Prettier, Eslint

## How to start

Install Docker. Here is an example how it can be done for Windows [link](https://docs.docker.com/desktop/install/windows-install/)

```bash
# start local product-feedback API server
npm run up

# start app
npm start

# clean up
npm run down
```
